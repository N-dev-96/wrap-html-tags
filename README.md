# Wrap HTML Tags

Extension for [Phoenix Code](https://phcode.io) that wraps selected text with HTML tags.

## Shortcuts

| Shortcut | Result |
|----------|--------|
| `Ctrl+B` | `<strong>text</strong>` |
| `Ctrl+I` | `<em>text</em>` |

## Installation

1. Open Phoenix Code
2. Go to Extensions
3. Install from GitHub: `tvoje-jmeno/wrap-html-tags`
